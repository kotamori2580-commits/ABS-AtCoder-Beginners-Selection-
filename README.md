# テンプレ 
````
## [問題ID] 問題タイトル
### 🔗 問題リンク
https://atcoder.jp/contests/abs/tasks/abc086a_a

### 💡 アプローチ・考察
- 入力値 a, b の積が偶数か奇数かを判定する。
- 実際に掛け算をして、2で割った余りを見る。
### 🛠 実装 (Python)
```python
# ここにコードを貼り付ける
a, b = map(int, input().split())

if (a * b) % 2 == 0:
    print("Even")
else:
    print("Odd")
```
````

# このテンプレを使うメリット

1.  **「アプローチ」を書く習慣がつく**
    * コーディング面接では「コードを書く前に、どう解くかを口頭で説明すること」を求められます。その練習になります。
2.  **「計算量（オーダー）」の意識**
    * `$O(N)$` とか `$O(1)$` を書く欄を作っています（分からなければ空欄でOK）。これを意識しているだけで、エンジニアとしての基礎力が段違いに見えます。
3.  **検索性が高い**
    * 後で「あの書き方、どの問題でやったっけ？」と思った時に、自分のリポジトリ内で検索できます。
